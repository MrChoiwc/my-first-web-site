CSS

Cascading Style Sheets (CSS) is style sheet language used for describing the presentation of a document written in a markup language such as HTMl. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS is designed to enble the sepration of presentation and content, including layoutm colors, and fonts. This separtion can inprove content accessibility; provide more flexibilty and control in the specification of presentation characteristics: enable multiple web pages to share formatting by specfying the relevant CSS in a seprate .css file, which reduces complexity and repetition in the structural content; and enable the .css file to be cached to improve the page load speed betwwen the pages that share the file and its formatting
